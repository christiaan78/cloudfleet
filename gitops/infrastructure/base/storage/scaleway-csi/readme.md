## Create secret in Vault 
Create the following secret in Vault: 
k8s/platform/scaleway-csi-secrets

With the following keys:
SCW_ACCESS_KEY = your Scaleway API access key
SCW_SECRET_KEY = your Scaleway API secret key
SCW_DEFAULT_PROJECT_ID = your Scaleway project ID (UUID)
SCW_DEFAULT_ZONE = nl-ams-1 (or whatever zone you want)
